# Modelagem e Manipulação de Banco de Dados – Senac

Projetos acadêmicos desenvolvidos no **Curso Técnico em Desenvolvimento de Sistemas (Senac)**,
na unidade curricular **Auxiliar na Modelagem e Manipulação de Banco de Dados**.

## Conteúdos abordados
- Fundamentos de banco de dados relacionais
- Modelagem conceitual e Modelo Entidade-Relacionamento (MER)
- Linguagem SQL (DDL e DML)
- Consultas SQL, incluindo JOINs
- Organização, integridade e consistência dos dados

## Estrutura do repositório
- **Atividade 1:** Diagrama de Entidade-Relacionamento (DER)
- **Atividade 2:** Criação do banco de dados e estruturas (DDL)
- **Atividade 3:** Inserção, manipulação e consultas de dados (DML)

> Projetos acadêmicos com dados fictícios, desenvolvidos exclusivamente para fins educacionais.
